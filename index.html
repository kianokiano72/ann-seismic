Overview

Why predict the fundamental period?

The fundamental period of vibration governs the lateral forces a building must resist. Design codes estimate it with simple empirical formulas based mostly on building height — but these were calibrated on foreign building stock and tend to be inaccurate for the reinforced-concrete (RC) frame buildings common in Algeria.

This thesis trains an artificial neural network (ANN) to predict the two fundamental periods Tx / Ty from six geometric features. On an independent test set the ANN reaches positive R² in both directions where the code formulas collapse to negative R², and it cuts the prediction error (RMSE) roughly in half versus RPA 99/2003.

Training pipeline

  1. 01Clean — remove 5 non-typical buildings (aberrant T/H ratio) → 96 buildings.
  2. 02Split — 80 / 20 train–test, with an inner 15% validation split.
  3. 03Standardize — StandardScaler on inputs and targets.
  4. 04Train — up to 1000 epochs, early stopping (patience 50), LR plateau decay.
  5. 05Ensemble — average 5 models (seeds 42–1000) for the final prediction.

Note on generalization

The metric cards above report the training fit (R² Train Tx = 0.900, Ty = 0.800), which shows the network learned the training data well.

On the independent held-out test set (20 buildings the model never saw during training): R² Test Tx = 0.381 · R² Test Ty = 0.375. The train/test gap of 0.519 is expected with a dataset of only 96 buildings — collecting more data would reduce it. Even so, the ANN still clearly outperforms the RPA 99/2003, Eurocode 8 and ASCE 7-16 formulas on this same test set.

This tool is intended for preliminary estimation only.

Scope & limitations

  • Valid for typical RC frame buildings: H ≈ 8.5–64 m, 2–21 storeys.
  • Trained on 96 buildings — a small dataset, hence the train–test gap (0.519).
  • Buildings with an atypical T/H ratio fall outside the model's range.
  • Intended as a research/decision-support tool, not a code substitute.
